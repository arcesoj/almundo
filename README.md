### How to install :  

=======
1. Clone project `git@github.com:arcesoj/react-native-challenge.git`
2. Run Server 
    1. `cd almundo/almundoAPI/`
    2. run `npm install`
    3. run `nodemon server.js`
3. Run Application
    1. open new terminal `cd almundo`
    2. run `npm install`
    3. run `cd ios`
    4. run `pod update`
    5. run `react native run-ios`
